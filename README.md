## Project Setup

1. Create virtual environment.

```bash
python -m venv .venv
```

2. Activate the environment.

```bash
source .venv/Scripts/activate
```

3. Install dependencies.

```bash
pip install -r requirements.txt
```

4. Run script as a module.

```bash
python -m cider_genai.scrape.pipeline
```
